# JavaScript

- JavaScript는 웹 브라우저에서 동작하는 유일한 프로그래밍 언어입니다.

## JavaScript 특징

- 동적 타입의 언어입니다.
- 프로토타입 기반의 객체지향 언어입니다.
- JavaScript는 함수형, 명령형, 프로토타입 기반 객체지향 프로그래밍을 지원하는 멀티 패러다임 프로그래밍 언어입니다.
